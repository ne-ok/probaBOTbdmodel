# probaBOTbdmodel


**1) Таблица БИЛЕТЫ (Tickets):**

- Id - номер записи (автоматически)

- EventName - что за билет ("Билет в аквапарк - Утренний")

- Date - на какую дату ("2025-12-25")

- TotalTickets - всего билетов (100)

- SoldTickets - продано (25)

- Price - цена (1500.00)

- Type - тип ("online")
**Пример: "Утренний билет на 25.12.2025, 75 штук осталось, 1500 руб"**

2) Таблица РЕСТОРАН (RestaurantReservations):

- Id - номер записи
  
- Date - дата ("2025-12-25")
  
- TimeSlot - время ("18:00-20:00")
  
- AvailableTables - свободно столиков (3)
  
- TotalTables - всего столиков (10)
**Пример: "25.12.2025 с 18:00 до 20:00 свободно 3 столика из 10"**

**3) Таблица АКВАПАРК (WaterparkOccupancy):**

- Id - номер записи
  
- Date - дата ("2025-12-25")
  
- TimeSlot - время ("10:00-12:00")
  
- CurrentVisitors - сейчас людей (45)
  
- MaxCapacity - максимум людей (100)
**Пример: "25.12.2025 с 10:00 до 12:00 в аквапарке 45 человек из 100"**

**4) Таблица ТАРИФЫ (Tariffs):**

- Id - номер записи
  
- Name - название ("Стандарт")
  
- Description - описание ("Вход на 2 часа")
  
- Price - цена (1500.00)
  
- Duration - время ("2 часа")
  
- Category - категория ("waterpark")
  
**Пример: "Стандартный билет - 1500 руб, на 2 часа, для аквапарка"**
